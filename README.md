# Cafe_Website_Frontend
A modern, responsive Cafe &amp; Restaurant Website built using HTML, CSS, and Bootstrap 5. This project showcases a complete frontend layout for a real-world cafe, including sections like Home, About, Menu, Gallery, Reviews, and Contact.

🚀 Features
🎨 Elegant UI with aesthetic color palette (cream, espresso, sage, oak)
📱 Fully responsive design using Bootstrap Grid
🧭 Sticky navigation bar with smooth scrolling
🖼 Image carousel for gallery section
🍽 Menu cards with food items and pricing
⭐ Customer review/testimonial section
📩 Contact form UI
⚡ Clean typography using Google Fonts

🛠 Tech Stack
HTML5 – Structure of the website
CSS3 – Styling and custom theme
Bootstrap 5 – Layout, responsiveness, and components
Google Fonts – Typography (Montserrat & Playfair Display)
